# CleanMind

**CleanMind** is a mobile application focused on digital detox and intentional content blocking.  
It helps users reduce access to pornographic content and distracting social media platforms through accountability, friction-based unlocks, and progress tracking.

Official domain (tentative):  
**cleanmindblocker.com**
